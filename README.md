# Lychee Website

Public website for Lychee Labs, rebuilt as a multi-page Next.js site for industrial software and intelligence positioning.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Local development

```bash
npm install
npm run dev
```

## Editable site constants

Update the following in [`lib/site.ts`](./lib/site.ts):

- founder name
- founder email
- Calendly URL
- LinkedIn URL
- company tagline

## Deploy

The repository is already linked to Vercel via `.vercel/project.json`.
